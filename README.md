### 21. RSA (шифрование)
**Тип:** Асимметричное шифрование

**Алгоритм шифрования:**
- **Ключи:** открытый `(e, n)`, закрытый `(d, n)`
- **Шифрование:** `C = M^e mod n`

**Принцип работы:**
1. Генерация простых `p`, `q`

3. Выбор `e` (взаимно простое с `φ(n)`)
4. Вычисление `d = e^(-1) mod φ(n)`
